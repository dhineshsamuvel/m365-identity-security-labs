## Objective
Administer Exchange Online mail services securely.

## Environment
Microsoft Exchange Online (Microsoft 365 tenant)

## Configuration Performed
- Created shared mailboxes
- Created distribution lists
- Configured mailbox permissions
- Performed message trace
- Reviewed anti-spam and anti-phishing policies

## Security Rationale
Proper mailbox configuration ensures controlled access and visibility into mail flow.  
Anti-phishing policies reduce email-based attack risks.

## Outcome
- Mailboxes and distribution lists configured
- Mail flow monitored
- Email security controls applied

## Validation
Mailbox access and message tracking functioned as configured.
